# lucky 插件 — 隐私权说明

**最后更新：2025-12-15**

## 背景
lucky 插件是一款面向公司内网的辅助工具，提供 AI 对话、快速划词取数等功能。为实现这些功能，插件会请求并使用若干浏览器权限。我们重视您的隐私与数据安全，以下说明了我们为何需要这些权限、会收集哪些信息以及如何存储与保护这些信息。

---

## 我们请求的权限与用途

1. 请求浏览器 `tabs`、`webRequest`、`webNavigation` 权限  
   - 用途：获取当前页面信息及部分公司内网请求相关信息，用以支持功能（例如上下文感知的 AI 对话、抓取或分析页面内容）。  
   - 存储：用于记录用户触发的相关操作数据（例如划词来源页面、操作时的页面 URL 或标识符），以便恢复上下文、排查问题或优化功能。

2. 请求 `storage`（本地存储）权限  
   - 用途：用于保存用户的配置和偏好设置（例如语言、快捷键、已登录状态、常用配置等），以提供持久化体验和个性化服务。  
   - 存储位置：信息保存在浏览器扩展的本地存储中，不会主动上传到外部服务器，除非为实现某些功能或在您明确同意下进行同步。

3. 请求 `scripting` 权限  
   - 用途：在相关页面注入脚本（比如划词采集脚本、快捷入口、浮窗等），帮助用户快速保存或提取想要的数据并提供即时交互体验。  
   - 范围：脚本仅在用户允许或插件功能触发时注入，且注入行为遵循最小权限原则，仅访问插件功能所需的页面内容。

## 我们会收集的用户活动记录
为改进和维护插件功能，我们会收集一些关于您使用插件的活动记录，例如但不限于：

- 打开插件的时间或频率  
- 点击登录 / 点击特定功能项的事件  
- 打开登录二维码的操作  
- 收藏账号或其他配置更改的记录

用途：这些数据用于产品改进、功能分析、错误排查与用户体验优化。我们不会将这些数据出售或共享给第三方机构或个人，除非：
- 基于法律或政府机关的强制要求；或
- 在获得您明确同意的情况下。
